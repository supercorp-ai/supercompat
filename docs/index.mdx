# Supercompat

Supercompat is an OpenAI-compatible TypeScript SDK that lets you use the **OpenAI Responses API** with any LLM provider.

## Quick Start

```bash
npm install supercompat
```

```typescript
import {
  supercompat,
  anthropicClientAdapter,
  completionsRunAdapter,
  prismaStorageAdapter,
} from 'supercompat/openai'
import Anthropic from '@anthropic-ai/sdk'

const client = supercompat({
  client: anthropicClientAdapter({ anthropic: new Anthropic() }),
  runAdapter: completionsRunAdapter(),
  storage: prismaStorageAdapter({ prisma }),
})

// Use the standard OpenAI Responses API
const response = await client.responses.create({
  model: 'claude-sonnet-4-6',
  input: 'Hello, world!',
})
```

## Architecture

Supercompat uses three types of adapters:

- **Client Adapter** — Which provider to call (OpenAI, Anthropic, Google, etc.)
- **Run Adapter** — Which API to use (Completions, Responses, Azure Agents)
- **Storage Adapter** — Where state lives (Memory, Prisma, OpenAI, Azure)
